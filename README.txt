CERTIFICATES FOLDER
===================

Store all your certificate images/PDFs here.

HOW TO USE:
-----------
1. Save certificate files as PNG, JPG, or PDF
2. Use path format in admin panel: certificates/filename.png
3. Click "View Certificate" button to open in modal

EXAMPLE FILES:
- certificates/internshala-web.png
- certificates/aws-forage.pdf
- certificates/infosys-c.png

That's it! Upload files here and update links in admin panel (Ctrl+Shift+O).
