# Professional Portfolio – Entry-Level Data & Web Analyst

This project is a professional portfolio website that demonstrates my skills as an entry-level Data Analyst and Web Developer.

## Role & Focus
- **Target role:** Entry-Level Data Analyst / Web & Data Analyst
- **Goal:** Present data analytics skills and web development experience in a clean, professional format.

## Structure
The portfolio includes:
- **Hero section:** Clear headline, professional summary, and quick access to Resume, GitHub, and LinkedIn.
- **About Me:** Career-focused overview of my BCA background and interest in data analytics + web development.
- **Skills:** Organized into Data & Analytics, Web Development, and Tools.
- **Projects:** Data and web projects with problem, solution, tech stack, and result.
- **Experience & Internships:** Brief, professional highlights.
- **Certifications:** Relevant platforms like Forage, Infosys Springboard, and Internshala.
- **Contact:** Email, LinkedIn, and GitHub links.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Data Tools:** Excel, SQL, Power BI, Python (for data analysis projects)
- **Version Control:** Git & GitHub
- **Hosting (planned):** GitHub Pages or Netlify

## Design Principles
- Clean, minimal layout
- White background with dark, readable text
- Subtle blue accent for emphasis
- No heavy animations or bright colors
